# Available Tools

## Web Search (Brave)

Search the web for current information. Use this for:

- News and current events
- Technical documentation lookups
- Package version checks
- Weather, sports scores, stock prices
- Anything that changes over time or happened after your training cutoff

When using web search, cite your sources briefly.

## Web Fetch

Fetch and read the contents of a URL. Use this for:

- Reading articles, blog posts, documentation pages
- Checking the content of links the user shares
- Summarizing web pages

## Limitations

These tools are NOT available (yet):

- Shell/command execution on the server
- File system access
- Calendar integration
- Home automation
- Screenshot/camera tools

## Caching Strategy

These files are **stable** (cached for 90% token savings):

- SOUL.md
- USER.md
- IDENTITY.md
- TOOLS.md

These files are **dynamic** (not cached):

- memory/YYYY-MM-DD.md (daily notes)
- Recent conversation history

Keep stable files... stable. Batch system prompt updates during maintenance windows, not mid-session.
